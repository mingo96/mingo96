## Proyectos actuales

### Kobre (Cobre.app / Cobre.web)

Plataforma de finanzas personales con app móvil y backend propio.

- **Backend**: `Cobre.web` — Laravel 13 + PHP 8.3, API REST `/api/v1/*` para autenticación, envelopes, transacciones, estadísticas, bancos, créditos, reglas, ajustes, dispositivos y webhooks.
- **Frontend web**: Inertia + React 19 + TypeScript, Tailwind y Vite para landing pública, área autenticada y panel de administración.
- **Infraestructura**: MySQL 8, Redis 7, Docker, scripts de desarrollo y despliegue, pipelines CI/CD reales en GitHub Actions.
- **Integraciones**: Tink (agregación bancaria), RevenueCat (webhooks de suscripciones), FCM y observabilidad con Sentry, logging estructurado y `request_id` para trazabilidad.

Este proyecto refleja cómo diseño y mantengo un **producto completo**: dominio definido, arquitectura documentada (ARCHITECTURE, DOMAIN, DATABASE, API_CONVENTIONS), flujos de deploy y operación en producción.

---

### RutinApp

Plataforma para gestión de rutinas de entrenamiento con **panel web y backend dedicados**, orientada a uso real.

- **Stack**: Laravel + React para la parte web, Kotlin Compose + KMP en la capa mobile (en progreso), con base de datos relacional y pagos in‑app.
- Diseño pensando en escalabilidad: separación entre API y panel, organización por módulos de negocio y soporte para futuras integraciones (apps móviles, servicios externos).
